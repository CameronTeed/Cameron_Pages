About the Project

Built with C++ on the G++ compiler for linux. This aplication outputs captured network packets for a set duration of time to a txt file. Testing was done with python, by broadcasting packets to the network and capturing them with the Network Analyzer. The python script then compares the output of the application to the packets sent, and verifies that the application is working correctly.

Key Features

  • C++ Development with Linux
  • File I/O
  • Testing and Quality Assurance
  • Python Scripting
  • Version Control
  • MVC
  • Sockets

How it works

Uses the linux socket library to capture packets, and outputs them to a file. The application is run from the command line, and takes in a duration of time to capture packets for.

Design

The apllication is designed using the MVC design pattern. The model is the socket and frame proccessing, the view is the output file, and the controller controls the view and model.

Analyzing Packets

In order to capture packets, and output them to a file, we must understand how packets are structured. Packets are made up of an ethernet header, an ip header, a protocol header, a payload, and sometime a trailer. The headers contains information about the packet, such as the source and destination IP address, and the payload contains the data being sent.

Sockets

A raw socket setup to capture all data was created. The data is then extracted from the packet using the included files which contain the needed structs to extract the bits.

Testing

Unit tests were conducted using google test (gtest), integration and functional tests were conducted using python, where data packets were broadcasted and searched for in the output file created by the network analyzer. Valgrind was used to ensure there were no memory leaks.

Optimizing Performance

The application focuses on optimizing performance in capturing and processing network packets. Techniques such as efficient memory handling and low-level socket programming contribute to the overall speed and reliability of the network analyzer.
