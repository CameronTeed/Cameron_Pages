About the Project

Built with C++ on the G++ compiler for linux. This aplication outputs captured network packets for a set duration of time to a txt file. Testing was done with python, by broadcasting packets to the network and capturing them with the application. The python script then compared the output of the application to the packets sent, and verified that the application was working correctly.
